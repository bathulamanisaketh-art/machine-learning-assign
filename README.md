## Quick Start

```bash
# 1. Clone
git clone https://github.com/yourusername/neural-style-transfer
cd neural-style-transfer

# 2. Install
pip install torch torchvision matplotlib numpy

# 3. Run
jupyter notebook neural_style_transfer_tutorial.ipynb
```

Run all cells top to bottom. No external data required — images are generated in the notebook.  
For real VGG-19: the notebook downloads pretrained weights automatically (~548MB, one-time).

## What This Tutorial Covers

1. **Why CNNs encode content and style separately** — VGG-19 feature hierarchy
2. **Content loss** — MSE between deep feature maps at conv4_2
3. **Style loss** — Gram matrix: position-invariant feature co-occurrence
4. **Total loss** — alpha × content + beta × style
5. **Optimisation** — pixel gradient descent, L-BFGS vs Adam
6. **From-scratch PyTorch implementation** — full working code
7. **Alpha/Beta ratio experiment** — novel investigation of the content–style trade-off
8. **Evolution to real-time NST** — Johnson (2016), AdaIN (2017), StyleGAN
